# 互勉约拍App产品总结

---

## 产品名称（暂定）
**「HearthVale」** 或 **「墟里」**  
*（结合杭州地域属性，强化本地信任感）*

---

## 一句话定位
**杭州首个专注于“拍照互勉”的信任型社交平台——让想拍照的人和会拍照的人，安全、靠谱地约到一起。**

---

## 核心价值
**解决杭州互勉市场“想拍不敢拍、约了怕被骗”的信任危机。**

| 用户痛点 | 产品解决方案 |
| :--- | :--- |
| **不敢约**：担心人身安全，怕遇到骚扰或诈骗 | **信任体系**：实名认证+芝麻信用分接入+互评机制 |
| **约不成**：约好被放鸽子，拍了不给返图 | **履约保障**：电子协议模板+押金托管（可选） |
| **找不到**：风格不对路，沟通成本高 | **智能匹配**：风格标签+同城推荐+作品墙 |
| **没地方拍**：不知道杭州哪里好拍 | **场景地图**：杭州网红打卡点+周边约拍动态 |

---

## 目标用户

| 用户类型 | 具体画像 | 核心诉求 |
| :--- | :--- | :--- |
| **素人模特** | 杭州高校女生、白领，想拍好看照片但不想花钱 | 免费获得高质量照片，安全地约拍 |
| **新手摄影师** | 杭州摄影爱好者、陪拍师，需要练手和积累作品 | 积累作品集，找到愿意出镜的模特 |
| **摄影发烧友** | 想尝试新风格（汉服、JK、胶片）的资深玩家 | 找到风格匹配的创作搭子 |

---

## 核心功能（MVP版本）

### 1. 信任基石
- **实名认证**：手机号+身份证双重认证，未认证无法发起约拍
- **芝麻信用分接入**：信用分650分以上用户获得“信用标识”
- **双向互评**：约拍完成后互评，建立杭州本地互勉信用榜

### 2. 约拍工具
- **电子协议模板**：内置标准约拍协议，明确返图数量、时间、肖像权使用范围
- **押金托管（可选）**：一方可发起押金托管，完成约拍后自动释放（解决“放鸽子”和“不返图”）
- **聊天留痕**：所有沟通记录云端保存，纠纷时可作为证据

### 3. 匹配引擎
- **风格标签**：日系、汉服、胶片、情绪、JK、复古等20+标签
- **作品墙**：摄影师和模特展示代表作，按风格、地点筛选
- **同城推荐**：优先推荐距离近、信用高的用户

### 4. 杭州专属
- **杭州拍照地图**：标注太子湾、西湖、满觉陇、拱墅运河公园等30+热门打卡点，每个点聚合附近约拍动态
- **季节主题运营**：樱花季、桂花季、银杏季推出主题约拍活动

---

## 冷启动策略（杭州路线图）

### 阶段一：种子用户（第1-2个月）
- **高校切入**：与浙大、国美、浙传的摄影社团合作，举办“校园互拍大赛”
- **陪拍师入驻**：通过小红书私信杭州陪拍师，邀请入驻，给予“首批认证”标识
- **种子用户目标**：500名摄影师 + 1000名模特

### 阶段二：场景引爆（第3-4个月）
- **杭州拍照地图上线**：联合杭州咖啡馆、摄影基地作为“安全约拍点”，首次约拍可在此见面
- **季节限定活动**：结合西湖桂花季/银杏季，推出“杭州最美打卡点互勉挑战”
- **用户目标**：日活500+，月约拍订单1000+

### 阶段三：信任固化（第5-6个月）
- **杭州互勉信用榜**：每月发布杭州信用TOP100摄影师/模特
- **线下见面会**：组织“杭州互拍搭子沙龙”，让线上信任延伸到线下
- **模式跑通后**：复制到成都、南京、长沙等“网红+高校”城市

## 盈利模式（远期）

| 阶段 | 盈利方式 | 说明 |
| :--- | :--- | :--- |
| **免费期** | 不收费 | 积累用户，建立杭州互勉生态 |
| **探索期** | 押金托管手续费 | 每笔押金托管收取1%-2%作为支付通道费 |
| **成熟期** | 摄影师会员 | 摄影师可付费开通“高级会员”，获得更多曝光和约拍机会 |
| **拓展期** | 商业约拍抽成 | 为专业摄影师开放“商业约拍”频道，平台抽成5%-10% |
